~ dot product distributes addition

Top Edit

? A*(B+C) = A*B + A*C

A*(B+C) = x_A*(x_B + x_C) + y_A*(y_B + y_C) = x_A*x_B + x_A*x_C + y_A*y_B + y_A*y_C = (x_A*x_B + y_A*y_B) + (x_A*x_C + y_A*y_C)

~ equivalence between geometric and analytic dot product definitions

Top Edit

? |a|*|b|*cos(a,b) = x_a*x_b + y_a*y_b

OX = OA * cos AOB = OA * (AB2 - OA2 - OB^2) / (2 * OA * OB) = &

AB2 = (xB - xA)2 + (yB - yA)2 OA2 = xA2 + yA2, etc so

& = ((xB - xA)2 + (yB - yA)2 - xA2 - yA2 - xB2 - yB2) / 2 * (xB2 - yB2) = (xAxB + yAyB) / xB2 - yB2 => OX * OB = OA * OB * cos AOB = xAxB + yAyB

~ transformation of coordinates in rotation

Top Edit

xA' = (xA / cos α) * cos (α + γ) = (xA * cos α * cos γ - xA * sin α * sin γ) / cos α = (xA * cos α * cos γ - xA * sin α * sin γ) / cos α = xA * cos γ - yA * sin γ

yA' = (yA / sin α) * sin (α + γ) = (yA * sin α * cos γ + yA * cos α * sin γ) / sin α = xA * sin γ + yA * cos γ

A' = A * | cos γ sin γ | | -sin γ cos γ |
